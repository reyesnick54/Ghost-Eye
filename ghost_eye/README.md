# Abraxas GhostEye

Abraxas GhostEye is a controlled-environment WiFi sensing demonstration platform.

The prototype is designed to show how WiFi Channel State Information, RF sensing, and inference models may be integrated into a mobile-facing interface for authorized research, defense, emergency response, and controlled test environments.

## Current Status

This repository currently contains:

- Third-party WiFi sensing and CSI research repositories under `third_party/wifi-sensing/vendor/`
- A GhostEye prototype backend under `ghost_eye/backend/`
- A mobile-friendly browser demo under `prototypes/mobile-demo/`

## Safety and Governance

GhostEye demonstrations must be:

- Authorized
- Consent-based
- Conducted in controlled environments
- Used only with test hardware and approved participants
- Not used for covert surveillance, unauthorized monitoring, or unlawful tracking

## Prototype Roadmap

1. Simulated mobile demo
2. Live CSI ingestion adapter
3. Local inference pipeline
4. Secure API layer
5. Native iOS / Android app wrapper
6. Field-testable controlled demonstration package
